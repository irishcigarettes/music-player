## Quick Start

### Option 1: Simple (Double-click)
Just double-click `start-server.bat` to start the local server, then open your browser to `http://localhost:3000`

### Option 2: PowerShell
Run this command in PowerShell:
```powershell
powershell.exe -ExecutionPolicy Bypass -File server.ps1
```

Then open your browser to: `http://localhost:3000`

### Option 3: Direct File
You can also just open `index.html` directly in your browser (some features may work better with a local server)

## Features

- **Y2K Aesthetic**: Bright gradients, sparkles, animations, and that early 2000s web vibe
- **Interactive Blog Posts**: Like posts, leave comments, and interact with content
- **Character Profile**: Display your character's stats and information
- **Gallery**: Placeholder gallery with modal views
- **Smooth Animations**: Cursor trails, sparkles, and hover effects
- **Local Storage**: Your likes and comments are saved locally

## How to Use

1. Start the server using one of the methods above
2. Navigate between sections using the top navigation
3. Like blog posts by clicking the heart button
4. Leave comments on any blog post
5. Explore the gallery and character information

## Customization

You can easily customize:
- Blog post content in `index.html`
- Character information in the About section
- Colors and styling in `styles.css`
- Interactive features in `script.js`
